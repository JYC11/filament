## Bugs Found & Fixed

3 bugs found during initial run, all fixed in this session:

1. **MCP `filament_task_close` accepted non-task entities** (Sev: Medium)
   - `task_close` on an agent entity succeeded instead of returning a validation error
   - Fix: Added `entity_type != "task"` guard in `mcp.rs` (matching CLI behavior)

2. **MCP `filament_message_send` accepted nonexistent recipients** (Sev: Medium)
   - Sending a message to a non-existent agent name succeeded silently
   - Fix: Added recipient validation (entity must exist and be type=agent) in `mcp.rs`

3. **`release_reservation` silently succeeded for nonexistent IDs** (Sev: Low)
   - Store-level `release_reservation()` did `DELETE WHERE id = ?` without checking `rows_affected()`
   - Fix: Check `rows_affected() == 0` and return new `ReservationNotFound` error variant
